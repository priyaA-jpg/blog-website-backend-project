# Website Backend Project

## Description
This project contains the backend implementation of a content-based website. 
It allows users to create, read, edit, copy, and delete posts. 
The system manages server-side logic and database operations.

## Technologies Used
- Node.js
- MongoDB
- REST API
- Git & GitHub

## Features
- Create new posts
- Read and display content
- Edit existing posts
- Delete posts
- Copy content functionality

## Purpose
This project demonstrates backend development skills including API handling, 
database integration, and server-side logic.
